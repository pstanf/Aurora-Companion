REST TAB — Meditation tracks by healing frequency
=================================================

The Rest tab plays one MP3 per Solfeggio frequency. Name each file with
the Hz number followed by "hz" (lowercase):

  174hz.mp3
  285hz.mp3
  396hz.mp3
  417hz.mp3
  432hz.mp3
  528hz.mp3
  639hz.mp3
  741hz.mp3
  852hz.mp3
  963hz.mp3

IMPORTANT: Each track should be longer than 10 minutes (the longest
Rest session). That way the music never runs out mid-session.

If a file is missing, the app uses a soft built-in tone at that
frequency as a fallback.

Recommended sources (free commercial use):
  • Pixabay — https://pixabay.com/music/
  • Mixkit — https://mixkit.co/free-stock-music/mood/meditative/

Keep a note of each track name and license page for your records.
